### Плагины

Плагины представляют из себя пользовательские модули, которые могут полностью изменить игровой процесс.

Сами плагины делятся на 2 типа: игровые и расширения.


#### Игровые плагины

Такой плагин может быть всего один в сессии он полностью описывает весь игровой процесс.

#### Плагины расширения

Расширения используются для игровых плагинов (или даже для других плагинов расширения) для расширения возможностей разработки, можно считать, что это своего рода библиотеки для расширения возможностей. Игровое правило при разработке может ссылаться на несколько плагинов расширений, и при создании новых сессий плагины расширения подгружаются автоматически, как зависимости для конкретного игрового плагина.
